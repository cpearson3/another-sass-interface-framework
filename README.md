# Bootsmooth
## Another Responsive SCSS Framework

Bootsmooth is a minimal and responsinve SCSS framework. It is built with Gulp.

Components:

- Responsive Grid
- Nav bar
- Buttons
- Forms
- Tables

### Building

    git clone https://github.com/cpearson3/bootsmooth.git
    bower install
    npm install
    gulp watch
