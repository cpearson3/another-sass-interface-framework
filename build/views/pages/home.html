Bootsmooth

A Respsonsive App Framework

Built with Angular, Sass, Gulp & Stuff


Github Project

Bootsmooth is a combination of two things:

  1. A simple, responsive CSS framework built with Sass
  2. A set of reusable UI components built with Angular JS

Used together, these pieces help developers quickly lay out and build front-end apps for today's devices.
Gett started today:

git clone https://github.com/cpearson3/bootsmooth.git
bower install
npm install
gulp watch
